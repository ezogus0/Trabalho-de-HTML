DELTA PCS Memória Ram, quanto maior a frequência melhor é a memória, a capacidade vai depender do que você vai fazer, se for usar o pc para renderizar animações 3D ou algo do tipo, é recomendado ter pelo menos 32GB, agora, se for só para jogar mesmo é recomendando usar pelo menos 16GB.
Outra informação importante é se a memória ram é ddr4 ou ddr5, se ela for ddr4 a placa mãe e o processador tem que ser compatível com pelo menos ddr4, agora, se ambos não forem compatíveis nem adianta instalar por que não irá funcionar.
Uma última informação para escolher a memória certa é se atentar sobre a frequência, isso você terá mais informações na página sobre os processadores.
